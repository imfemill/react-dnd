# [React Dnd](https://react-dnd.github.io/react-dnd/about)

> clone the repository

```bash
git clone https://github.com/imfemill/react-dnd.git
```

> go to directory

```bash
cd react-dnd
```

> install dependencies

```bash
# install node_modules
npm install
# or
yarn
# or
bun install
```

> start the application

```bash
npm run dev
# or
yarn run dev
# or
bun run dev
```
